# Architecture Decision Records (ADRs)

This directory contains the significant architectural decisions made in
FinVeritas Contabolsa. Each ADR follows the Nygard format:

1. **Status** - proposed, accepted, deprecated, superseded
2. **Context** - forces shaping the decision
3. **Decision** - what we decided
4. **Consequences** - positive and negative trade-offs

new ADR explaining the change.
